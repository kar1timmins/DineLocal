## Principles

1. **Server Components by Default** - Add `'use client'` only when needed
2. **Type Safety** - TypeScript for all, proper interfaces
3. **Accessibility** - WCAG 2.2 AA minimum
4. **Performance** - Lazy load, optimize images
5. **Composition** - Small, focused components
